# EventCore

Bu proje, ASP.NET Core ve JavaScript kullanılarak geliştirilmiş bir etkinlik takvimi ve rezervasyon sistemi içerir.

## Özellikler

- Etkinlikleri listeleyebilme
- Etkinlik detaylarını görüntüleme
- Etkinliklere rezervasyon yapabilme
- Günlük, haftalık ve aylık takvim görünümü

## Kullanılan Teknolojiler

- ASP.NET Core
- JavaScript
